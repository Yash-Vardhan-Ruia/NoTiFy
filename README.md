## 🌟 About NoTiFy  

NoTiFy is a **free, open-source, and feature-rich** alternative to Notion that helps you **take notes, plan meals, manage tasks, and collaborate effortlessly**.  

Designed with an **intuitive and engaging UI**, NoTiFy makes productivity easy for **students, developers, professionals, and teams**. It provides a **flexible workspace** where you can manage everything in one place—**for free**! 🚀

## 🛠️ Tech Stack 

### **Frontend**  
- **Next.js (App Router)**  
- **Tailwind CSS**  
- **TipTap (Rich Text Editor)**  

### **Backend**  
- **Next.js + Supabase**  
- **PostgreSQL (via Supabase)**  

## 🚀 Getting Started  

### **Prerequisites**  
Ensure you have the following installed:  
- [Node.js](https://nodejs.org/) (LTS version recommended)  

### **Installation**  

#### 1⃣ Clone the repository:  
```sh
git clone https://github.com/Yash-Vardhan-Ruia/NoTiFy.git
```

#### 2⃣ Install dependencies:  
```sh
npm install
```

#### 3⃣ Run the development server:  
```sh
npm run dev
```
